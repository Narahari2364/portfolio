# Portfolio Website

A modern, animated portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Art-first Design**: Minimal, clean interface that puts artwork front and center
- **Fluid Animations**: Smooth, premium animations inspired by slime-style/fluid UI motion
- **Fully Responsive**: Optimized for all screen sizes
- **Performance Optimized**: Built with Next.js App Router for optimal performance
- **Accessible**: Follows best practices for accessibility

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- Replace artwork images in `components/Gallery.tsx` with your own
- Update hero text in `components/Hero.tsx`
- Modify colors and styling in `tailwind.config.ts` and `app/globals.css`
- Add your own artwork data to the `artworks` array in `components/Gallery.tsx`

## Build for Production

```bash
npm run build
npm start
```
